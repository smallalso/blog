小亦的blog

# 技术栈

- **Framework**: [Astro](https://astro.build/)
- **Deployment**: [cloudflare](https://pages.cloudflare.com/) 
- **Styling**: [Tailwind CSS](https://tailwindcss.com)

## 本地开发

This application requires Node.js v18.17+.

```bash
pnpm install
pnpm run start or pnpm dev
```
